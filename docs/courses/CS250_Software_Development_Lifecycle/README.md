
# CS 250 – Software Development Lifecycle (May–Jun 2026)

Coursework for **CS 250 – Software Development Lifecycle**, taking place **May – June 2026**.  This course examines processes and methodologies for developing software projects.

Expect to include:

* Analyses of different development models (e.g., Waterfall vs Agile).
* Project management artifacts such as requirement specifications, Gantt charts, and retrospectives.
* Group project deliverables demonstrating the complete development lifecycle.
